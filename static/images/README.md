# Open Graph Image - IMPORTANT!

## Upload Your OG Image Here

**Filename**: `og-image.jpg`

**Required Specs**:
- **Size**: 1200x630px (Facebook/LinkedIn standard)
- **Format**: JPG or PNG
- **Max File Size**: Under 1MB

## Tools to Create:

1. **Canva** (easiest) - https://canva.com
   - Use "Facebook Post" template (1200x630)
   - Customize with your info

2. **Figma** (for designers)
   - Create custom design
   - Export as JPG

3. **Online Generators**
   - https://www.bannerbear.com/
   - https://og-playground.vercel.app/
